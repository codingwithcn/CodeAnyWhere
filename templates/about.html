What is CodeAnyWhere

CodeAnyWhere is a free web application that allows you to access your development folders, and change the files from where ever you are at the moment.

How I came up with the idea

It happened when I had to leave my workspace suddenly with no warning. Giving me no chance to save files on another computer or even on Github. I was extremely annoyed by this. Which forced me to think of better and more time efficient ways to do what I love. Leading to CodeAnyWhere.

How to get started

Assuming you are reading this, I am guessing you already know how to get started. But if you are still confuse I will gladly explain.

  • First you would either have to pip install this project from Github or directly to github and get this.
  • Next you will need to open your command line and then head inside this project location
  • Once in, download a virtual env using the command:"python -m pip venv env "
  • Next you will have to activate this virtual environment.
  • Once activated you can now run the program with the command: "python main.py "
  • Wait for it to load. Once loaded, you should see a statement saying: "Running on (the ip address its running on)". Copy and paste that Ip address into you web browser of your choosing.
  • You should be taking to the home page. Then you would have to create an account. You can do this two ways: 1. Head over to the database and then to the user table and add in a new account. If you are not sure what you are doing I do not advise doing that, because a simple mistake could mess up the database and the program. 2. You can simply sign up using the website. On the home page there should be a "SignUp ". Click on that and you would be taking to the sign up page.
  • Once you have create an account, head over to Login and you can now access you folders from any computer on your local network.
  • If you would like to access your files world wide. You will need to learn about port forwarding, and open your ports to incoming connections. But be warned this process is dangerous because it means you will be open to attacks. And people might be able to access you information.
